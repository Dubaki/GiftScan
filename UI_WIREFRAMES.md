# UI Screens

## 1. Dashboard (Scanner)
- **Таблица:** Список подарков. Колонки: Иконка, Название, Мин. цена (Buy), Макс. цена (Sell), Spread (Profit %).
- **Фильтры:** По проценту профита, по цене (Low/High).
- **Хайлайты:** Подсвечивать зеленым строки, где профит > 10% с учетом комиссий.

## 2. Gift Detail (Карточка товара)
- График изменения цены за 24ч / 7д.
- Стакан цен (Order Book) с разных площадок.
- Кнопка **"Flash Buy"** (Только для Premium) -> открывает диплинк на оплату.

## 3. Escrow Create
- Форма: "Я отдаю" (выбор из инвентаря) <-> "Я хочу получить" (выбор из каталога или USDT).
- Калькулятор комиссии сервиса.
